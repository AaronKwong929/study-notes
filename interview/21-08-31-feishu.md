```js
function getNumber(n) {
  const set = new Set();
  const res = new Array(n);
  for (let i = 0; i < n; i++) {
    let num = getNum(n);
    while (set.has(num)) {
      num = getNum(n);
    }
    res[i] = num;
    set.add(num);
  }
  return res;
}
```

-这个时间复杂度多少？

-满打满算 O(n^2)

-有没有好一点的解决方法

```js
function getNumber(n) {
  const res = [];
  for (let i = 0; i < n; i++) {
    res[i] = i;
  }
  res = res.sort((a, b) => (Math.random() > 0.5 ? a - b : b - a));
  return res;
}
```

```js
function getNumber(n) {
  const res = [];
  for (let i = 0; i < n; i++) res[i] = i;

  for (let i = array.length - 1; i > 0; i--) {
    let j = Math.floor(Math.random() * (i + 1));
```
